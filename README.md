#  SunCart – Summer Essentials Store

> A modern, vibrant summer eCommerce platform for seasonal products — sunglasses, outfits, skincare, beach accessories, and more.

##  Live URL

[https://suncart.vercel.app](https://suncart.vercel.app)

---

##  Key Features

-  Browse summer products with elegant product cards
-  Authentication (Email/Password + Google OAuth) via BetterAuth
-  Protected Product Detail pages — login required
-  Fully responsive: mobile, tablet & desktop
-  User profile with update functionality
-  Hero carousel with summer sale banners
-  Summer Care Tips section
-  Top Brands showcase
-  Unique luxury beach resort design aesthetic
-  Lottie animations for enhanced UX
-  Toast notifications for auth feedback

---

##  NPM Packages Used

| Package | Purpose |
|---|---|
| `next` | React framework with App Router |
| `better-auth` | Full-stack authentication |
| `better-sqlite3` | SQLite database for BetterAuth |
| `react-hot-toast` | Toast notifications |
| `lottie-react` | JSON-based animations |
| `animate.css` | CSS animation library |
| `react-icons` | Icon library |
| `swiper` | Hero slider/carousel |
| `tailwindcss` | Utility-first CSS |
| `daisyui` | Tailwind component library |

---

Open [http://localhost:3000](http://localhost:3000)

---

##  Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + DaisyUI
- **Auth**: BetterAuth
- **Database**: SQLite (via better-sqlite3)
- **Deployment**: Vercel
